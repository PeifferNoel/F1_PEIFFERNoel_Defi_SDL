-Différences SDL / SDL2

     On peut citer deux principales différences entre le SDL et le SDL2:
	SDL est plus vieux et est supporté par moins d'interfaces que SDL2


-Pourquoi SDL2_Image ?

     Nativement, SDL2 ne supporte que les images en BMP, on utilise donc SDL2_Image pour les images dans d'autres formats
